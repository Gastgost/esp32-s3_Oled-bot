# ESP32-S3 OLED Image Bot

Telegram бот для конвертации изображений в битмапы для OLED дисплея ESP32-S3.

## Функции
- Конвертация любых изображений в монохромный формат 128x64
- Генерация HEX массивов для ESP32
- Просмотр превью результата

## Использование
1. Отправь боту изображение
2. Получи превью и HEX массив
3. Используй массив в коде ESP32
